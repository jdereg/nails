nails
==========

Nails is a web application framework built with java-util, json-io, n-cube, and the json-command-servlet.
```
<dependency>
  <groupId>com.cedarsoftware</groupId>
  <artifactId>nails</artifactId>
  <version>1.0.0</version>
</dependency>
```
